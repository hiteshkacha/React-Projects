# React Projects

A collection of React projects — built to learn, experiment, and ship things that actually work.

---

## What's in here

This repo isn't a single app. It's a running record of React work — different ideas, different scopes, different problems worth solving. Each project lives in its own directory with its own setup.

---

## Structure

```
/
├── project-name/
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── README.md
├── another-project/
│   └── ...
└── README.md
```

Every project is self-contained. Navigate into the folder, install dependencies, and run it.

---

## Getting started with any project

```bash
cd project-name
npm install
npm run dev
```

No global setup required. Each project manages its own dependencies.

## Stack I work with

- **React** — functional components, hooks, modern patterns
- **Vite** — fast builds, no config headaches
- **Tailwind CSS** — utility-first, when it fits
- **React Router** — for anything that needs navigation
- **TypeScript** — on projects where it earns its place

## Why this repo exists

Not every project deserves its own standalone repo. This is a space to build things without the overhead — experiments that turn into real features, UI components worth keeping, ideas explored properly.
